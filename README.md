Instructions to install new version:
1. After code changes, run `python -m PyInstaller --onefile --noconsole --distpath . --name PersonalPasswordManager PasswordManager.py`
2. Backup data
3. Go to C:\Program Files (x86)\PersonalPasswordManager and run the uninstaller
4. Open NSIS and run InstallerScript.nsi
5. Run the generated PersonalPasswordManagerInstaller.exe in Github/Personal-Password-Manager
